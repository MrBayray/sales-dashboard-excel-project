# 📊 Sales Dashboard Excel Project

## 🚀 Overview
This project is an interactive Excel Sales Dashboard designed to analyze and visualize business sales data. It transforms raw data into meaningful insights using Pivot Tables, Charts, and Slicers.

---

## 🎯 Objectives
- Clean and structure raw sales data
- Analyze sales by product, region, category, and time
- Build key performance indicators (KPIs)
- Create an interactive dashboard for business insights

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts (Column Charts)  
- Slicers  

---

## 📊 Dashboard Features
- Total Sales by Product  
- Sales by Region  
- Sales Distribution by Category  
- Sales Trend Over Time (Monthly)  
- Interactive filtering using slicers  

---

## 📈 Analysis Process
1. Data was cleaned and structured in Excel  
2. Pivot tables were created for different business views  
3. Charts were built to visualize key insights  
4. Slicers were added for interactivity  
5. Dashboard was arranged for clear storytelling  

---

## 💡 Key Insights
- Certain products generate higher revenue than others  
- Regional performance varies significantly  
- Category breakdown shows differences in product demand  
- Sales trends help identify performance over time
